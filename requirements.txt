# ================================================================
# INSTALLATION INSTRUCTIONS
# ================================================================

# 1. Install Python 3.8 or higher from https://www.python.org/downloads/
#    ✅ IMPORTANT: Check "Add Python to PATH" during installation!

# 2. Install FFmpeg (required for audio extraction):
#    - Download from: https://www.gyan.dev/ffmpeg/builds/
#    - Extract and add to PATH, OR place ffmpeg.exe in this folder

# 3. Install Python dependencies:
#    Open terminal in this folder and run:
#    pip install -r requirements.txt

# 4. (Optional) For GPU acceleration:
#    - Install NVIDIA CUDA Toolkit 11.8
#    - Install PyTorch with CUDA:
#      pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

# ================================================================
# USAGE
# ================================================================

# METHOD 1: Using batch file (recommended)
# - Double-click: run_transcription.bat
# - Select model and settings from menu
# - Press [R] to run

# METHOD 2: Direct Python execution
# - Edit settings in generate_transcripts.py
# - Run: python generate_transcripts.py
